# Módulo 08 - Projeto: O Painel de Métricas (Dashboard)

## Descrição do Projeto
Você vai desenhar (no papel ou slide) o Dashboard ideal para um Time Ágil.

## Cenário
Seu time tem uma TV na parede (ou um monitor virtual). O que deve estar lá para que todos saibam se estão ganhando ou perdendo o jogo?

## Requisitos do Projeto

O Dashboard deve conter no MÁXIMO 4 KPIs (Key Performance Indicators). Se tiver 20, ninguém lê.
Escolha 1 métrica de cada categoria:
1.  **Velocidade/Fluxo:** (Ex: Throughput, Burn-down).
2.  **Qualidade:** (Ex: Bugs abertos, Cobertura de Testes).
3.  **Negócio/Satisfação:** (Ex: Vendas, NPS, Happiness Metric do time).
4.  **Atenção/Gargalo:** (Ex: Dias sem deploy, Idade média dos itens em progresso).

## Entregável
Um desenho do layout desse dashboard explicando POR QUE você escolheu cada uma das 4 métricas.
