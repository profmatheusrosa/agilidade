# Módulo 07 - Projeto: Simulando um "PI Planning" de Bolso

## Descrição do Projeto
Você vai organizar 3 "times fictícios" para entregar um jantar de Ação de Graças (ou Natal) completo para 30 pessoas.

## Cenário
O projeto é "O Grande Jantar".
*   **Time A (Cozinha Quente):** Assados e guarnições.
*   **Time B (Cozinha Fria/Bebidas):** Saladas, sobremesas e drinks.
*   **Time C (Salão):** Decoração, mesas, serviço.

## Execução

### 1. Visão do Produto (Roteiro)
O que queremos? "Um jantar inesquecível, servido às 20h em ponto, com tudo quente."

### 2. Identificação de Dependências (O Pulo do Gato)
Liste 3 dependências críticas entre os times.
*   *Ex:* O Time C só pode arrumar a mesa se os pratos estiverem limpos (mas quem lava?).
*   *Ex:* O Time Quente precisa saber que horas os convidados chegam (Controlado pelo Time C) para colocar o peru no forno.
*   *Ex:* O Time Frio precisa de espaço na geladeira, mas o Time Quente ocupou tudo com carne marinando.

### 3. O Quadro de Programa
Desenhe uma linha do tempo das 14h às 20h. Aloque as grandes tarefas de cada time. Onde as linhas se cruzam?

## Entregável
Uma lista das dependências identificadas e como você as resolveu (ex: "Time A comprou um isopor com gelo para liberar a geladeira para o Time B").
